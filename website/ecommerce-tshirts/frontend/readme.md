# Frontend Service

Interactive web interface for the Learning-Performance-Observability-Stack platform.

## Overview

Modern, responsive web frontend built with Astro and TailwindCSS, providing:

- **Interactive API playground** for testing endpoints
- **Comprehensive documentation** browser
- **Architecture diagrams** and guides  
- **Observability dashboard** links
- **API reference** documentation
- **Educational content** for learning performance and observability

## Technology Stack

- **Framework:** Astro 5.16.0
- **Styling:** TailwindCSS
- **UI Components:** Custom React components
- **Deployment:** Nginx (production) / Node.js (development)
- **Build Tool:** Vite

## Quick Start

```bash
# Development
yarn dev

# Build
yarn build

# Preview
yarn preview
```

Access at: http://localhost:4321 (dev) or http://localhost:4000 (Docker)

## Documentation

For complete frontend documentation, see the inline README in this directory or visit the deployed site at `/documentation`.
